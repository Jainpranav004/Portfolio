# Plant Disease Detection

## Overview
An AI-powered plant disease detection system built using **MobileNetV2**, **PyTorch**, and **TensorFlow/Keras** for real-time image-based disease classification.
The application allows users to upload plant leaf images and automatically predicts diseases using trained deep learning models.

![Image](/projects/plant_clasifier.png)
---

## Features
- Real-time plant disease prediction
- Deep learning image classification
- MobileNetV2-based architecture
- Streamlit-powered web interface
- Supports multiple crop disease classes
- PyTorch and Keras model support


## Tech Stack
- Python
- Streamlit
- PyTorch
- TensorFlow / Keras
- MobileNetV2
- NumPy
- PIL


## Model Details
- MobileNetV2-based image classification
- 38-class disease detection model
- EfficientNet-based alternate model
- Trained on plant leaf disease datasets
- Optimized for real-time inference

- `app.py` → Streamlit application
- `disease_model.pth` → PyTorch model weights
